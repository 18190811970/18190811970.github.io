django操作

Posted on 2018-12-29 | In Django 
Django ***
虚拟环境
安装: pip install virtualenv
使用: virtualenv --no-site-packages -p D:\python3\python.exe name
    --no-site-packages: 表示创建的虚拟环境为纯净的环境，不安装有其他的库
    -p: 表示虚拟环境中的python版本

pip使用
    pip list: 插看安装的库
    pip install xxx: 安装xxx

激活虚拟环境
    Windows: 直接执行activate命令
    Mac/Linux/Ubuntu: 直接执行source activate命令

退出虚拟环境
    Windows/Mac/Linux/Ubuntu: 直接执行deactivate

Django项目

创建项目: django-amdin startproject 项目名称
创建应用: python manage.py startapp 应用名称
启动
    启动命令: python manage.py runserver。默认IP为127.0.0.1  默认端口为8000
    修改启动端口: python manage.py runserver 端口
    修改IP和端口: python manage.py runserver IP:端口
    IP参数: 如果为0.0.0.0表示任何人都可以通过公网IP访问django项目
    端口PORT参数: 如果端口设置为80，表示该端口可以不用写

Django模式

M
    模型层: 定义模型和数据库中的表之间的关联关系
V
    视图层: 定义业务逻辑
T
    模板: HTML页面

数据库准备工作 ***

settings.py中配置 
    USER参数: 访问用户
    PASSWORD参数: 密码
    HOST参数: 访问数据库的地址
    PORT参数: 访问数据库的端口
    NAME参数: 数据库名
    'OPTIONS':{'isolation_level':None}
安装pymysql
    使用pymysql连接数据库: 因为python3没有MySQLdb驱动，无法直接连接MySQL
    在工程目录的__init__.py文件中定义  pymysql.install_as_MySQLdb()
迁移
    第一次迁移: python manege.py migrate
    除开第一次迁移
        生成迁移文件: python manage.py makemigrations
        执行迁移文件: python manage.py migrate
插入管理员账号
    python manage.py createsuperuser

模型层

概念
    objects: 管理器，默认模型的属性
    ORM(objects relationship mapping): 对象关系映射
创建
    create()方法: 模型名.objects.create(字段1=值,字段2=值)
    save()方法: 模型对象.save()

    def add_stu(request):
        实现插入数据
        实现的第一中方式:
        Student.objects.create(s_name='小明')
        实现的第二中方法:
        stu = Student()
        stu.s_name = '黄老邪'
        stu.s_gender = 1
        stu.save()
        return HttpResponse('创建学生成功')

删除
    delete()方法: 模型名.objects.filter(条件).delete()

    def del_stu(request):
        1.获取删除对象 ， filter（条件）
        2.实现删除方法 ， delete
        Student.objects.filter(id=2).delete()
        return HttpResponse('删除成功')

修改or更新
    update()方法: 模型名.objects.filter(条件).update(字段1=值,字段2=值)
    save()方法: 模型对象.save()
    update()和save()的区别: 是否更新使用auto_now约束条件的字段

    def up_stu(request):
        # 实现更新
        # 1.实现更新的数据 filter（条件）
        # 2.实现更新的方法 update（）
        # 更新的第一中方法，只更新数据 ，不更新时间
        # Student.objects.filter(id=1).update(s_name='风清扬')
        # 更新的第二中方法 ,即更新时间也更新数据
        stu = Student.objects.filter(id=1).first()
        stu.s_name = '杨过'
        stu.save()
        return HttpResponse('更新学生信息')
查询
    filter()方法,查询满足条件的信息: 模型名.objects.filter(条件)
         # 查询ID=1的学生信息 ，filter()
        stu = Student.objects.filter(id=1)
    all()方法: 查询所有
        Student.objects.all()
    get()方法: 查询指定的条件
        # 1.get()取唯一的一个对象
        # 2.get(条件)条件必须成立
        stu = Student.objects.get(id=4)
        print(stu)

    first()方法: 取出结果中的第一个对象
        stu = Student.objects.filter(id=1).first()

    exclude()方法: 查询出不满足条件的信息: 模型名.objects.exclude(条件)
        stus = Student.objects.exclude(s_gender=0) 

    order_by()方法: 升序order_by('id')  降序order_by('-id')
        stus = Student.objects.order_by('-id')

    values()方法: 取查询对象中的字段
        stus = Student.objects.all().values('s_name')

    exists(): 判断结果是否存在,存在返回True,否则返回False
        a = Student.objects.filter(s_name='小张').exists()

    count()方法: 计算结果的条数
        b = Student.objects.filter(s_gender=1).count()

    pk: 主键（就是id）
        stus = Student.objects.filter(id__in=[1,2,3,4,5,6])
        stus = Student.objects.filter(pk__in=[1,2,3,4,5,6])

    运算符
        语法:  字段__contains
        contains: 包含
            stus = Student.objects.filter(s_name__contains='黄')

        startswith: 以什么开头
            stus = Student.objects.filter(s_name__startswith='黄')

        endswith: 以什么结束
            stus = Student.objects.filter(s_name__endswith='邪')

        in: 在某个范围之内
            stus = Student.objects.filter(id__in=[1,2,3,4,5,6])

            gt：大于,gte:大于等
            stus = Student.objects.filter(s_age__gte=18, s_age__lt=20)
            lt：小于 lte :小于等于


    聚合
        导入：from django.db.models import Max,Avg, Min, Sum, Count
            age_avg = Student.objects.all().aggregate(Avg('s_age'))
            age_sum = Student.objects.all().aggregate(Sum('s_age'))

        语法: 模型名.objects.all().aggregate(Sum(字段))
    Q
        与1: Q(条件1), Q(条件2)
             stus = Student.objects.filter(Q(s_age__gte=18) ， Q(s_age__lt=20))
        与2: Q(条件1) & Q(条件2)
             stus = Student.objects.filter(Q(s_age__gte=18) & Q(s_age__lt=20))
        或: Q(条件1) | Q(条件2)
             stus = Student.objects.filter(Q(s_age__gte=18) | Q(s_age__lt=20))
        非: ~Q(条件1)
            stus = Student.objects.filter(~Q(s_age__gte=18))
    F
        用于比较两个字段,可使用+,-符号
        模型名.objects.filter(wuli__gt=F('math')-10)
            查询物理成绩大于数学的学生
            stus = Student.objects.filter(wuli__gt=F('math'))
            print()
